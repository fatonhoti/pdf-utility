## Getting Started

The following instructions should get a demo going on your PC.

### Installation

1. Clone the repo

   ```
   git clone https://github.com/fatonhoti/pdf-utility.git
   cd pdf-utility
   ```
3. Install relevant packages
   ```
   pip install -r requirements.txt
   ```
4. Run the game
   ```
   python util.py
   ```
